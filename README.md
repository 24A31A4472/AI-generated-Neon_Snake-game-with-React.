# 🐍 NEON_SNAKE_OS

A retro-futuristic Snake game experience combined with a built-in music player, designed with a cyberpunk neon UI.
Built using **React** and enhanced with **Google AI Studio & Gemini** for rapid development and UI iteration.

---

## 🚀 Live Demo

🔗 https://neon-snake-os-228495768384.us-west1.run.app/

---

## ✨ Features

* 🎮 Classic Snake Game with modern neon aesthetics
* 🎧 Integrated Music Player (play/pause, controls, volume)
* ⚡ Smooth UI with retro cyberpunk design
* 🧠 AI-assisted development using Gemini
* ☁️ Deployed on Google Cloud Run

---

## 🛠️ Tech Stack

* **Frontend:** React.js
* **Styling:** CSS (Neon/Cyberpunk theme)
* **AI Tools:** Google AI Studio, Gemini
* **Deployment:** Google Cloud Run
* **Version Control:** Git & GitHub
